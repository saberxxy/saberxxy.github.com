初识Julia（一）

发表于 2018-08-11 | 分类于 Julia
初识Julia（一）这是关于Julia的第一篇文章
  有一门语言叫Julia，它十分小众，以至于在TIOBE上只排到了第50名，和近年来飞速上升的Python相比，几乎可以用微不足道来形容。然而，最近Julia发布了1.0版本，这个消息迅速轰动了圈内。
  最初认识这门语言，还是在知乎上，当时看到一篇文章：
  https://zhuanlan.zhihu.com/p/41723707
  这篇文章里的一段话吸引了我的眼球：

> 我们想要一种十分自由的开源语言，同时兼具 C 语言的速度和 Ruby 语言的灵活度。我们想要一种同像性语言，具有像 Lisp 一样真正的宏，也有像 Maltlab 一样浅显易懂的数学符号。它是像 Python 一样有用的通用编程语言，像 R 语言一样便于统计的语言，像 Perl 一样自然的字符串处理语言，像 Matlab 一样强大的线性代数语言，也是像 shell 一样的「胶水语言」。它简单易学，却能让严苛的黑客们为之倾心。我们希望它兼具交互性和可编译性。


  这，这，这简直就是上天的恩赐！它瞬间点燃了我，让我无论如何，工作多忙，都必须抽出时间来接触它，使用它。我迫不及待的点开了官网：
  https://julialang.org/
  在Documentation中看到了这样的介绍：



不愧是集诸多语言的优势于一身，这正是我想要的！！！！
