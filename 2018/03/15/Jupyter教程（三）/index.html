Jupyter教程（三）

发表于 2018-03-15 | 分类于 Python
本篇主要讲一下Jupyter如何更换主题

  几天的使用之下，发下Jupyter确实好用，但有些时候实在看着有点眼瞎，于是乎萌生了换主题的念头。
1、先执行命令pip install jupyterthemes安装，如果网不好，会比较坑，因此直接到pypi下载，地址如下：传送门
2、下载完后，切换到文件的目录，直接安装。

3、输入命令jt -l，先看看都有些什么主题，发现主题有些少，为什么呢，因为并没有更新到最新版本的jupyterthemes，于是乎，卸掉旧版本，重新下载安装新版本。

4、更新的过程中可以发现，用一下清华大学的镜像源，速度非常之快，极好。具体使用方法就是如下命令：


5、安装完再看一下，主题确实多了不少

6、使用命令jt -t 主题名称，就可以切换主题了
7、jt -r 可以恢复默认样式

ps：安装完jupyterthemes要重启一下jupyter，然后更换主题才能生效。
