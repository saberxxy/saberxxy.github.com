再游圆明园

发表于 2019-07-02 | 分类于 诗词
岁在己亥端阳，于圆明三园之内，看残垣，观山水，怀古迹，品洞箫，寄诗情，畅心扉，颇多情愫，填词数阙。

  岁在己亥端阳，于圆明三园之内，看残垣，观山水，怀古迹，品洞箫，寄诗情，畅心扉，颇多情愫，填词数阙。


西江月
残山剩水几重，残垣断壁几处，楼阁馆榭付尘土，金银珠玉无数。
钟鸣鼎食之家，过眼繁华荒芜，雨打芭蕉人如簇，荒台野草如故。

声声慢
西风春柔，翠叶低窗，倩问谁持金锸。云山坞里，铁马金戈檐牙。有道是多情客，无情絮、飘飘洒洒。冷声里，陌上车如流，薰风撷花。
已是南阁曲断，天涯故人老，难成彩画。福海洞天，丝竹管弦嘲哳。凤阙龙楼仍在，雁归时、栖尽寒鸦。看湖上，花非花、那年柳下。

小重山
海岳开襟气象新，南秀接北渚，玉观音。濂溪香雪对旧楹。眉月凉，窗外晓寒轻。
韶景茂育亭。荷风回廊下，草凋零。鱼跃鸢飞枭鸟凭。佳山水，雅音有谁听。

浣溪沙
长春湖上九曲桥，仙人承露蒙尘嚣，洞天秀色忽缥缈。
浪里行舟摇桂棹，鹧鸪声断衡阳箫，昆岗玉碎香兰笑。

蝶恋花
醉别西楼锁秋千，人语长风，芙蓉泪如烟。方壶胜境留此间，不知今夕是何年。
寒衣未去卷珠帘，好梦惊回，余花落别院。休将春水细细看，阳关一曲柔肠断。

定风波
九州清晏长春馆，婆娑画舫渡水还。碧桐院外云岑亭，高宗，只与素笺寄象管。
西峰秀色小庐山，蹉跎光阴杭州湾。今朝春来须爱惜，文宗，一醉梦醒君不见。

钗头凤
荒塚留，残台旧，满园春色皇洲柳。廓然意，静寄庄。一缕东阳，几处流光。凉，凉，凉。
云也过，雨乍收，九州万方近古楼。得月处，鹤斋堂。喜鹊绕梁，何种思量。茫，茫，茫。
